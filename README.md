# Daily NHL Intelligence Report

This project is a fully automated daily data pipeline that ingests raw NHL game data, transforms it into advanced hockey analytics metrics using dbt, and publishes a clean HTML intelligence report every morning. Built with Apache Airflow, dbt Core, BigQuery, and Python, the system demonstrates end-to-end data engineering practices used by professional hockey analytics organizations. The pipeline runs on GCP within free tier limits and generates metrics including Corsi percentages, expected goals, high-danger chances, and player performance indicators.
